How was md2blog made?

October 25, 2021

md2blog is built on top of Metalsmith, an "extremely simple, pluggable static site generator", along with the following Metalsmith plugins:

  • metalsmith-broken-link-checker
  • metalsmith-collections
  • metalsmith-discover-partials
  • metalsmith-drafts
  • metalsmith-express
  • metalsmith-filemetadata
  • metalsmith-layouts
  • metalsmith-metadata
  • metalsmith-rootpath
  • metalsmith-taxonomy
  • metalsmith-watch

md2blog also uses the following libraries (unrelated to Metalsmith):

  • commander
  • dot2svg-wasm
  • Handlebars
  • highlight.js
  • JSTransformer
  • Less
  • Marked
  • route-parser

Libraries used for testing:

  • cheerio
  • Mocha
